Jenkins 用户文档的这一部分包含一系列的入门教程帮你使用 Jenkins 以自动化的方式开始构建你的应用程序。

如果你是一个想要提高对持续集成(CI) / 持续交付 (CD) 概念的理解的开发者, 或者你已经对这些概念很熟悉 但不知道如何使用Jenkins去实现, 那就从这些教程开始学习吧。

== 入门教程

本教程展示如何使用 Jenkins 涉及到基于特定的技术栈的CI/CD
概念的基础。

选择与你的技术栈有关或你最熟悉的一个教程:

* link:build-a-java-app-with-maven[使用 Maven 构建 Java 应用]
* link:build-a-node-js-and-react-app-with-npm[使用 npm 构建 Node.js 和 React 应用]
* link:build-a-python-app-with-pyinstaller[使用 PyInstaller 构建 Python 应用]

本教程展示 如何使用 Jenkins 的关键特性 (比如
link:/doc/book/blueocean/[Blue Ocean]) 帮助实现 CI/CD
进程来构建你的应用:

* link:create-a-pipeline-in-blue-ocean[使用 Blue Ocean 构建流水线]

== 高级教程

该教程展示 Jenkins 更高级的特性和如何管理你的更复杂更灵活的流水线项目。

* link:build-a-multibranch-pipeline-project[构建多分支流水线项目]
